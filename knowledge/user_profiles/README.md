# User Profiles — Per-User Knowledge Storage

Each user of Gaya gets their own profile directory. This is where personal lessons, preferences, and project notes live.

## How to Create Your Profile

1. Copy `_template/` to a folder named after you: `cp -r _template your-name`
2. Edit `profile.md` with your details
3. Add lessons to `lessons/` as you learn them
4. Add project notes to `projects/`

## The Profile Stays Local

Your profile is yours. It's in `.gitignore` by default — it won't be pushed to the repo unless you explicitly add it.

If you want to share a lesson with the community:
1. Copy it to `knowledge/community/submissions/`
2. Open a Pull Request
3. See `templates/KNOWLEDGE_CONTRIBUTION.md`
